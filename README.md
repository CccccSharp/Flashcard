# TCAS Vocab Vault

เว็บแฟลชการ์ด/ควิซคำศัพท์สำหรับสอบ TCAS / TGAT / A-Level
ดึงคลังคำศัพท์ตรงจาก **Google Sheets ของคุณ (แบบ CSV)** ไม่มีระบบล็อกอิน ไม่มีฐานข้อมูลภายนอก —
ความคืบหน้า (กองการ์ดล้างแค้น) และหมวดหมู่ที่คุณเพิ่ม จะถูกบันทึกไว้ใน**เบราว์เซอร์เครื่องนั้นๆ เท่านั้น**
(ใช้ `localStorage` ของเบราว์เซอร์ ไม่ต้องมี backend หรือบัญชีใดๆ)

---

## 1. รันบนเครื่องตัวเอง

ต้องมี [Node.js](https://nodejs.org) เวอร์ชัน 18 ขึ้นไป

```bash
npm install
npm run dev
```

เปิด `http://localhost:5173` ในเบราว์เซอร์

## 2. Deploy ขึ้นเว็บจริง

ไม่ต้องตั้งค่าอะไรเพิ่มเลย (ไม่มี environment variable ไม่ต้องมี Firebase project) เลือกทางใดทางหนึ่ง:

### ตัวเลือก A: Vercel หรือ Netlify (แนะนำ ง่ายที่สุด)
1. อัปโค้ดขึ้น GitHub repo
2. เข้า [vercel.com](https://vercel.com) หรือ [netlify.com](https://netlify.com) → New Project → เลือก repo นี้
3. Build command: `npm run build`, Output directory: `dist`
4. กด Deploy — เสร็จแล้ว ได้โดเมนใช้งานได้ทันที

### ตัวเลือก B: GitHub Pages / Firebase Hosting / โฮสต์ static ไฟล์อื่นๆ
```bash
npm run build
```
จะได้โฟลเดอร์ `dist/` เอาไฟล์ในนั้นอัปโหลดขึ้นโฮสต์ static ไฟล์ที่ไหนก็ได้

## 3. เพิ่มคลังคำศัพท์จาก Google Sheets

เปิดเว็บ → กดไอคอนเฟือง ⚙️ ที่หน้าแรก

1. เปิด Google Sheets ของคุณ → **File → Share → Publish to web**
2. เลือกชีตที่ต้องการ → เลือกรูปแบบ **Comma-separated values (.csv)** → **Publish**
3. คัดลอกลิงก์ที่ได้ → วางในช่อง "ลิงก์ CSV ที่เผยแพร่แล้ว" ในหน้าตั้งค่า พร้อมตั้งชื่อหมวดหมู่ (ไทย/อังกฤษ) แล้วกด "เพิ่มหมวดหมู่"
4. ทำซ้ำได้เรื่อยๆ — เพิ่มได้กี่ชุด CSV ก็ได้ แต่ละชุดจะกลายเป็นหนึ่งหมวดหมู่บนหน้าแรก
5. กดปุ่มรีเฟรช (วงกลมลูกศร) ที่แต่ละหมวดหมู่ได้ทุกเมื่อ เพื่อดึงข้อมูลล่าสุดจากชีตอีกครั้ง (เผื่อคุณแก้คำศัพท์ในชีตทีหลัง)

**คอลัมน์ที่ต้องมีในชีต (แถวแรกเป็นหัวคอลัมน์):**

| word | pos | thai | synonyms | antonyms | example | exampleTh | choice1 | choice2 | choice3 | choice4 | correctIndex |
|------|-----|------|----------|----------|---------|-----------|---------|---------|---------|---------|--------------|
| ambiguous | adj. | คลุมเครือ | vague;unclear | clear;precise | The instructions were ambiguous. | คำสั่งคลุมเครือ | คลุมเครือ | รวดเร็ว | ราคาแพง | อ่อนโยน | 0 |

- `synonyms` / `antonyms` คั่นด้วย `;`
- คอลัมน์ `choice1-4` และ `correctIndex` (0 = choice1, 1 = choice2, ...) เว้นว่างได้ถ้าคำนั้นไม่ต้องการโหมดควิซ — จะเหลือแค่โหมดแฟลชการ์ดของคำนั้น

## ข้อจำกัดที่ควรรู้

- **ไม่มีระบบบัญชี** — ถ้าเปิดเว็บนี้จากอุปกรณ์อื่น หรือล้างข้อมูลเบราว์เซอร์ (clear browsing data) กองการ์ดล้างแค้น + หมวดหมู่ที่เพิ่มไว้จะหายไป (ต้องเพิ่มลิงก์ Google Sheets ใหม่ แต่คำศัพท์ในชีตเองไม่หายเพราะเก็บอยู่ใน Google Sheets ของคุณ)
- ลิงก์ Sheet ต้องเป็นแบบ **Publish to web → CSV** (ไม่ใช่ลิงก์แชร์ปกติแบบ "Anyone with the link") เพราะเว็บต้องดึงข้อมูลแบบ public ไม่ผ่าน login

## โครงสร้างไฟล์

```
tcas-vocab-vault/
├── src/
│   ├── App.jsx        ← หน้าตาแอปทั้งหมด (Home, การ์ด, ควิซ, ตั้งค่า)
│   ├── storage.js      ← บันทึก/อ่านข้อมูลจาก localStorage ของเบราว์เซอร์
│   ├── main.jsx
│   └── index.css
├── .gitignore
└── package.json
```

แก้ไขคำศัพท์ตัวอย่าง (built-in ที่มากับเว็บ) ได้ที่ตัวแปร `CATEGORIES` ด้านบนของ `src/App.jsx`
ส่วนคำศัพท์ที่มาจาก Google Sheets จะถูกดึงสดทุกครั้งที่เปิดเว็บ (พร้อม cache สำรองไว้ใน localStorage เผื่อดึงไม่ได้ชั่วคราว เช่น อินเทอร์เน็ตหลุด)
